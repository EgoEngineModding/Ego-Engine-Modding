Requirements

  • .NET 8.0 Desktop Runtime (x64 or x86 depending on your OS)

Installation

To install EXTRACT everything from the zip file to a desired location, and run the Ego JPK Archiver.exe file.

App Information

This app lets you export and import resources in JPK archives for EGO engine games.

Usage

After running the app click File -> Open and select a JPK file to edit.

Now you may export the files to a folder, edit them, and import them back in before saving the JPK archive.

When creating a JPK from a folder, the app will first attempt to import the files listed in manifest.jpk.txt. If the manifest doesn't exist then all files from the folder will be imported in ascending order.
